# PayPulse – Smart QR Payment & Digital Wallet App

PayPulse is a modern **digital wallet and QR payment mobile application** designed to simplify financial transactions, expense tracking, and digital investments.

The application allows users to **scan QR codes, send and receive money, manage wallet balances, analyze spending patterns, and invest in digital gold** through a secure and intuitive interface.

---

## Overview

PayPulse provides a **complete financial management ecosystem** that helps users manage their money efficiently through real-time insights and smart features.

The system integrates:

- QR based payments
- Wallet management
- Financial analytics
- AI driven insights
- Digital gold investment
- Transaction history tracking

The goal of the project is to provide a **simple, secure, and intelligent digital payment experience.**

---

## Features

### Dashboard

The dashboard provides a quick overview of the user's financial status.

- Available wallet balance
- Savings balance
- Credit balance
- Pulse financial score
- Monthly growth indicators
- Privacy toggle for hiding balance
- Light / Dark theme toggle

---

### Wallet Operations

#### Add Money
Users can add money to their digital wallet for transactions.

#### Send Money
Allows secure transfer of funds to other users.

#### Receive Money
Users can receive payments directly into their wallet.

#### Split Payments
Divide expenses among multiple users for group payments.

---

### QR Code Payment Scanner

The **Scan feature** enables quick and secure payments using QR codes.

Use cases:

- Merchant payments
- Peer-to-peer transfers
- Instant checkout payments

---

### Digital Gold Investment

PayPulse allows users to buy and sell **digital gold** directly through the application.

Features include:

- Gold purchase tracking
- Gold sale transactions
- Investment monitoring
- Financial history tracking

---

### Transaction History

The system maintains a complete record of financial transactions.

Users can:

- View transaction details
- Search transactions
- Filter by category
- Track income and expenses
- Monitor wallet top-ups

---

### Financial Analytics

The analytics section provides financial insights and spending analysis.

Features:

- Monthly budget tracking

- Subscription detection
- Financial behavior insights

### AI Insights

The application generates **AI based financial suggestions**.

Examples:

- Identify highest spending categories

### Security Features

PayPulse ensures strong security through:

- Biometric authentication (Fingerprint / Face ID)

### Smart Saving Tools

#### Round-Up Auto Save

Automatically saves small amounts during transactions.

Example:

Transaction = ₹95  
Rounded to ₹100 → ₹5 saved automatically

This helps users develop consistent saving habits.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/paypulse-app.git
```

Move into the project directory:

```bash
cd paypulse-app
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```
